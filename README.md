# 東北大理学部でのPython+Jupyter notebook講習会

**開催日:**
2017年8月3日(木)-4日(金)  

**目的**  
光赤外撮像観測とそのデータ処理の基本概念、および実際のデータ処理の手法の基礎を学ぶ。
PythonとJupyter notebookを使ったインタラクティブな処理の基礎を身につけるとともに、
プログラミングの基本、Jupyter notebookで研究メモを残すスタイルについても身につける。
